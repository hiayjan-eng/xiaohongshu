# 收藏复活浏览器扩展 POC

这是一个 Manifest V3 浏览器扩展原型，用于验证“用户本人主动点击后，扫描小红书网页版收藏夹当前已加载 DOM”的技术可行性。

## 加载方式

1. 打开 Chrome 或 Microsoft Edge。
2. 进入扩展管理页：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. 打开开发者模式。
4. 选择“加载已解压的扩展”。
5. 选择本目录：`apps/extension`。

## 使用方式

1. 用户自己登录小红书网页版。
2. 手动打开自己的收藏夹页面。
3. 点击“收藏复活扫描 POC”扩展。
4. 点击“扫描当前可见卡片”或“轻滚动后扫描”。
5. 在 popup 中确认扫描结果。
6. 点击“导入收藏复活”，扩展会打开 Web MVP，并通过 `#extension-import=...` 把扫描清单交给 Web 本地导入。

## 边界

- 不登录账号。
- 不绕过验证码、风控或访问限制。
- 不读取不可见或未加载到 DOM 的内容。
- 不保存原帖全文，不复制图片/视频。
- 不上传扫描结果到第三方服务。
- 第一版只提取标题、原帖链接、封面 URL 和可见短文本。
